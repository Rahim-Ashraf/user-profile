### How to Start the project:
#### Installation
1. Clone the repository
2. Install dependencies:
    ```sh
   npm install
    ```
## Usage
1. Run the project:
    ```sh
    npm run dev
    ```
2. Build for production:
    ```sh
    npm run build
    ```
